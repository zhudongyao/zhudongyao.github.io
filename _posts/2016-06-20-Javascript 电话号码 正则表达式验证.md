以下为验证一串号码是否符合手机号规则的js代码：

```
var reg = /^1[3|4|5|7|8][0-9]{9}$/; //验证规则

var phoneNum = '15507621999';//手机号码

var flag = reg.test(phoneNum); //true
```

其中，验证规则是指第一个数字为“1”，第二个数字为“3，4，5，7，8”中的一个，第3~11个数字为0~9中的任意数字。
